# Commodity Futures Trading Commission

The Commodity Futures Trading Commission (CFTC) is the U.S. federal regulator for commodity futures and options markets. The CFTC publishes the weekly Commitments of Traders (COT) report and other public data through a Socrata Open Data API at publicreporting.cftc.gov, providing programmatic access to Legacy, Disaggregated, Traders in Financial Futures, and Supplemental Commodity Index Trader datasets.

## APIs

### CFTC Commitments of Traders SODA API
Programmatic access to CFTC COT weekly reports via the Socrata Open Data API at publicreporting.cftc.gov, covering Legacy, Disaggregated, TFF, and Supplemental Commodity Index Trader formats for both futures-only and combined futures-and-options positions.

**Human URL:** [https://publicreporting.cftc.gov/](https://publicreporting.cftc.gov/)

### CFTC Swap Data Repositories
The CFTC oversees Swap Data Repositories (SDRs) that collect and maintain swap transaction records as required by the Dodd-Frank Act.

**Human URL:** [Swaps Reports](https://www.cftc.gov/MarketReports/SwapsReports/index.htm)

#### Tags
- Dodd-Frank, Swaps, SDR, Reporting

### CFTC Bank Participation and Large Trader Reports
The CFTC publishes monthly Bank Participation reports and other aggregate large trader reports.

**Human URL:** [Bank Participation Reports](https://www.cftc.gov/MarketReports/BankParticipationReports/index.htm)

#### Tags
